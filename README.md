# Expense Project 

A complete **Django-based Expense Tracker** to manage personal finances efficiently.  
This project is perfect for a portfolio, showcasing user authentication, data visualization, and CRUD functionality.

---

##  Features
-  **Register, Login & Logout** – Secure user authentication  
-  **User-specific expenses** – Each user sees only their own data  
-  **CRUD operations** – Add, Edit, Delete, and View expenses  
-  **Pie Chart & Monthly Chart** – Visualize your spending habits  
- **Dashboard Cards** – Quick summary of income, expenses, and balance  

---

## 🛠 Technology Stack
- **Backend:** Django, Python  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite (can be replaced with MySQL/PostgreSQL)  
- **Charts & Visualizations:** Chart.js
